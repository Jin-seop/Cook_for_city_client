# Getting Started

**Using npm** :

1. 이 repository를 **다운로드** or **Git Clone**받으십시오.

2. 상위 디렉토리에 인스톨을 하십시오.

   ```
   $ npm install
   ```
3. 상위 디렉토리를 실행하십시오.

      ```
      $ npm start
      ```
